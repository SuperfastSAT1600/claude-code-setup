# AI Citability Scoring Skill Documentation

This comprehensive guide explains **GEO (Generative Engine Optimization)**, a framework for optimizing web content specifically for AI citation rather than traditional search rankings.

## Core Concept

The research indicates that language models preferentially cite passages meeting four criteria: "134-167 words long," self-contained passages, fact-rich content, and direct answers in opening sentences. This differs fundamentally from keyword-focused SEO.

## Scoring Framework (5 Categories)

**Answer Block Quality (30%):** Content should open sections with direct definitions using patterns like "X is..." rather than burying answers mid-paragraph.

**Passage Self-Containment (25%):** Individual blocks must stand alone without requiring prior context, explicitly naming subjects rather than using pronouns like "it."

**Structural Readability (20%):** Clean H1→H2→H3 hierarchies, question-based headings, short 2-4 sentence paragraphs, and strategic use of tables and lists improve AI parsing.

**Statistical Density (15%):** Specific percentages, dollar amounts, named studies, and exact counts significantly outperform vague quantifiers like "many."

**Uniqueness & Original Data (10%):** First-party research, proprietary datasets, original surveys, and unique frameworks make content an indispensable citation source.

## Key Findings

The research shows definition patterns increase citations by 2.1x, while adding statistics boosts citations by approximately 40%. Content with source attribution receives 20-25% more citations from certain AI systems.

Different platforms prioritize differently: Perplexity favors fact-dense content with 4-8 sources; ChatGPT typically cites 2-4 sources with recent dates; Claude values comprehensiveness.
